```python
import re
# 클리닝 함수 (file에 특수문자가 있을 경우 )
def clean_text(text):
    text = text.replace("\n", "")
    cleaned_text = re.sub('[\{\}\[\]\/?.,;:|\)*~`!^\-_+<>@\#$%&\\\=\(\'\"]',
                          '', text)
    return cleaned_text
```


